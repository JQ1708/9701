Question 1
Which one of the following has the same number of electrons as an alpha particle?
  • A H
  • B H+
  • C H2
  • D He
HINT: An alpha particle is a helium nucleus ($He^{2+}$).
Correct Answer: B

An alpha particle is a helium nucleus ($He^{2+}$) and it does not have any electrons.
A: H (1e)
B: H+ (0e)
C: H2 (2e)
D: He (2e)

Question 3
Which of the following elements has the largest second ionisation energy?
  • A O
  • B F
  • C Ne
  • D Na
HINT: Second IE removes an electron from the X+ ion. Look for the X+ ion that has a stable Noble Gas configuration.
Correct Answer: D

Second ionisation energy corresponds to $X^+(g) \rightarrow X^{2+}(g) + e^-$.
$Na^+$ has a stable noble gas configuration (2,8). Removing a second electron disrupts this stable octet, requiring the most energy.
